## 📖 Overview  
This is the frontend of a full-stack Recipe Management Web Application built using React.  
It allows users to browse, create, edit, and manage recipes with a clean and responsive UI.

---

## 🚀 Features  

- 🔐 User Authentication (Login / Signup)  
- 🍽️ View all recipes (Feed)  
- ➕ Add new recipes  
- ✏️ Edit existing recipes  
- 🗑️ Delete recipes  
- 👤 User profile with personal recipes  
- 📄 Detailed recipe view  
- 🌐 Integrated with REST API (ASP.NET Core backend)

---

## 🛠️ Tech Stack  

- Frontend: React.js  
- Styling: CSS / Bootstrap  
- API Calls: Axios  
- Routing: React Router  
- Deployment: Netlify  
- CI/CD: GitHub Actions

## ⚙️ Setup & Installation  

### -Clone repository
git clone https://github.com/ShameerNavas/RECIPE-Management-Frontend.git

### -Navigate to project
cd RECIPE-Management-Frontend

### -Install dependencies
npm install

### -Run project
npm start
